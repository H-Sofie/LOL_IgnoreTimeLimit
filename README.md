# LOL_IgnoreTimeLimit
Removed the time limit for League of Legends in China (anti-addiction system)

移除国区中英雄联盟的时间限制（防沉迷系统）

###### 新手 写的不好见谅

用法：将LOL_IgnoreTimeLimit放在'英雄联盟'文件夹内。文件夹目录类似于：
```
\英雄联盟(游戏目录)
    ├── Cross
    ├── Game
    ├── LeagueClient
    ├── Riot Client
    ├── Riot Client Data
    └── TCLS
```
然后运行即可

亦或者打开后选择至该路径也可以
